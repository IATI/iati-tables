===========
IATI Tables
===========

`IATI Tables <https://tables.iatistandard.org/>`_ transforms all the data published on the IATI Registry from XML files into relational tables,
and makes it available to users in multiple formats, e.g. PostgreSQL, CSV, and a `Datasette instance <https://datasette.tables.iatistandard.org/>`_.
